# HoshinoBot 原神KFC语音插件
能让星乃发出来自二次元的声音（雾）

## 安装方法

1. 安装ffmpeg
2. 将`genshinkfc`目录放入`hoshino/modules`目录中
3. 打开`hoshino/config/__bot__.py`，在`MODULES_ON`中加入`genshinkfc`并保存
4. 重启HoshinoBot
## 指令

|       指令       |    说明    |
| :--------------: | :--------: |
|       KFC        | 大小写均可 |
|     原神KFC      | 大小写均可 |
| 异世相遇尽享美味 |            |
